# Project Comet - API

## Running gRPC UI
### Linux Instructions
Install Go, gRPC UI
```sh
sudo apt install golang-go
go install github.com/fullstorydev/grpcui/cmd/grpcui@latest
```
Run gRPC UI
```sh
grpcui -plaintext localhost:5234
```
Find gRPC UI documentation [here](https://github.com/fullstorydev/grpcui).
